# Digital Art

Interactive visual scenes built with PixiJS 8 and React.

**[Live Demo](https://mbehrisch.github.io/digital-art/)**

## Scenes

- **Bouncing Shapes** — Circles and rectangles bouncing with collisions, proximity lines, and glow effects
- **Hexagon Grid** — Honeycomb color waves with ripple pulses synced to timing events
- **Bouncing Shape Game** — Click connected shapes to score points with a sniper scope cursor

## Tech

React, TypeScript, PixiJS 8, Vite, React Router
